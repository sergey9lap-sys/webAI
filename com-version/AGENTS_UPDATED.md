Use as inspiration level, not for copying:

- Cuberto

- Bruno Simon

## Before Implementing Any Section
1. Think of at least 3 layout concepts.
2. Compare them.
3. Choose the most unique and expressive option.
4. Avoid the first obvious SaaS solution.

## Preferred Design Direction
- Cinematic storytelling
- Scroll-driven experiences
- Layered depth
- Premium typography
- Unexpected transitions
- Strong visual composition

## Avoid
- Generic SaaS layouts
- Large empty sections
- Default Tailwind appearance
- Repetitive cards

## Content Rules
Never place technical, placeholder, or developer text on the website.

Forbidden examples:
- "Placeholder"
- "Section Title"
- "Feature Card"
- "Image will be added later"
- Technical notes visible to users

All temporary content must still look like real website content.
